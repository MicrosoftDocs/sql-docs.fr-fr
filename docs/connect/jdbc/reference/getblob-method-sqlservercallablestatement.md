# <a name="getblob-method-sqlservercallablestatement"></a>Méthode getBlob (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre BLOB JDBC désigné sous forme d’objet Blob dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int.md)|Récupère la valeur du paramètre BLOB JDBC désigné sous forme d’objet Blob dans le langage de programmation Java en fonction de l’index de paramètre.|  
|[getBlob (java.lang.String)](../../../connect/jdbc/reference/getblob-method-java-lang-string.md)|Récupère la valeur du paramètre BLOB JDBC désigné sous forme d’objet Blob dans le langage de programmation Java en fonction du nom de paramètre.|
