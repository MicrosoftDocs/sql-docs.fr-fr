# <a name="sqlserverexception-constructor-javalangstring-javalangthrowable"></a>Constructeur SQLServerException (java.lang.String, java.lang.Throwable)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Initialise une nouvelle instance de la classe [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) en fonction d’un objet **string** et d’un objet **jetable**.

## <a name="syntax"></a>Syntaxe  
  
```  
public SQLServerException(java.lang.String errText,
            java.lang.Throwable cause)
            
```  
  
#### <a name="parameters"></a>Paramètres  
 *errText*  
  
 Chaîne contenant le texte de l’erreur.
 
 *cause*  
  
 Objet jetable qui contient la cause de l’exception.
