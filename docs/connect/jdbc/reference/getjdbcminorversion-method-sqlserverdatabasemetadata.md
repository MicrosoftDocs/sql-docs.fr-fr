# <a name="getjdbcminorversion-method-sqlserverdatabasemetadata"></a>Méthode getJDBCMinorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère le numéro de version mineure JDBC pour ce pilote.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public int getJDBCMinorVersion()  
```  
  
## <a name="return-value"></a>Valeur de retour  
 **int** indiquant la version mineure de JDBC.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getJDBCMinorVersion est spécifiée par la méthode getJDBCMinorVersion de l’interface java.sql.DatabaseMetaData.
