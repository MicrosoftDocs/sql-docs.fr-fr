# <a name="updateshort-method-sqlserverresultset"></a>updateShort, méthode (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Met à jour la colonne désignée avec une valeur **short**.

|Nom|Description|  
|----------|-----------------|  
|[updateShort (int, short)](../../../connect/jdbc/reference/updateshort-method-int-short.md)|Met à jour la colonne désignée avec une valeur **short** en fonction de l’index de colonne.|  
|[updateShort (java.lang.String, short)](../../../connect/jdbc/reference/updateshort-method-java-lang-string-short.md)|Met à jour la colonne désignée avec une valeur **short** en fonction du nom de colonne.|
