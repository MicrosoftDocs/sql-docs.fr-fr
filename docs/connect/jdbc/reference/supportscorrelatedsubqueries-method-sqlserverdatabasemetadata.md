# <a name="supportscorrelatedsubqueries-method-sqlserverdatabasemetadata"></a>Méthode supportsCorrelatedSubqueries (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère les informations déterminant si cette base de données prend en charge les sous-requêtes corrélées.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public boolean supportsCorrelatedSubqueries()  
```  
  
## <a name="return-value"></a>Valeur de retour  
 **true** en cas de prise en charge. Dans le cas contraire, la valeur est **false**.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode supportsCorelatedSubqueries est spécifiée par la méthode supportsCorelatedSubqueries de l’interface java.sql.DatabaseMetaData.
