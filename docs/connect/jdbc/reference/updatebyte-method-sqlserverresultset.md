# <a name="updatebyte-method-sqlserverresultset"></a>updateByte, méthode (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Met à jour la colonne désignée avec une valeur **byte**.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom|Description|  
|----------|-----------------|  
|[updateByte (int, byte)](../../../connect/jdbc/reference/updatebyte-method-int-byte.md)|Met à jour la colonne désignée avec une valeur **d’octet** en fonction de l’index de colonne.|  
|[updateByte (java.lang.String, byte)](../../../connect/jdbc/reference/updatebyte-method-java-lang-string-byte.md)|Met à jour la colonne désignée avec une valeur **d’octet** en fonction du nom de colonne.|
