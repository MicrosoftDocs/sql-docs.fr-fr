# <a name="setcharacterstream-method-sqlservercallablestatement"></a>Méthode setCharacterStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|----------|-----------------|
