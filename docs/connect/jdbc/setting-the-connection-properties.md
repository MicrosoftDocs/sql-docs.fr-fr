# <a name="setting-the-connection-properties"></a>Définition des propriétés de connexion

[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

Les propriétés de chaîne de connexion peuvent être spécifiées de plusieurs manières :

- En tant que propriétés nom=valeur dans l’URL de connexion, lors de la connexion à l’aide de la classe DriverManager.
- En tant que propriétés nom=valeur Properties dans le paramètre *Properties* de la méthode Connect de la classe DriverManager.
- En tant que valeurs dans la méthode setter appropriée de la source de données du pilote. Par exemple :  
  
    ```java
    datasource.setServerName(value)  
    datasource.setDatabaseName(value)  
    ```  
  
## <a name="remarks"></a>Notes

Les noms de propriétés ne sont pas sensibles à la casse et les noms de propriétés dupliqués sont résolus dans l'ordre suivant :  
  
1. Arguments API (utilisateur et mot de passe)
2. Collection de propriétés.  
3. Dernière instance de la chaîne de connexion.
  
Par ailleurs, des valeurs inconnues sont autorisées dans les noms de propriétés et leurs valeurs ne sont pas validées par le pilote JDBC concernant la sensibilité à la casse.

Les synonymes sont autorisés et sont résolus dans l'ordre, tout comme les noms de propriétés dupliqués.

La bibliothèque ADAL peut être installée dans le cadre de l’installation de [Microsoft ODBC Driver pour SQL Server](../odbc/download-odbc-driver-for-sql-server.md) ou [Microsoft OLE DB Driver pour SQL Server](../oledb/download-oledb-driver-for-sql-server.md). Sinon, une erreur se produit (les propriétés de connexion doivent être spécifiées).  |

Cette propriété s’applique uniquement si « queryTimeout » est également défini sur la connexion. <br/><br/>Le pilote attend la valeur totale en secondes de **cancelQueryTimeout** + **queryTimeout** pour annuler la connexion et fermer le canal. <br/><br/>La valeur par défaut de cette propriété est -1 et le comportement consiste à attendre indéfiniment. |

| delayLoadingLobs<br/><br/>boolean<br/>["true" &#124; "false"]<br/><br/>true | Indicateur qui spécifie s’il faut ou non diffuser en continu tous les objets LOB récupérés de ResultSet.
