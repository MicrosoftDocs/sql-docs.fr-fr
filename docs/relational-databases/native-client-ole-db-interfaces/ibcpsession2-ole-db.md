# <a name="ibcpsession2-native-client-ole-db-provider"></a>IBCPSession2 (fournisseur Native Client OLE DB)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  L'interface IBCPSession2 est une extension à IBCPSession qui fournit une fonction membre qui est une alternative à l'appel de IBCPSession::BCPColFmt pour chaque colonne.  IBCPSession2 hérite de IBCPSession et ajoute une nouvelle méthode : [IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
