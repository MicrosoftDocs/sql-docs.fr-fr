# <a name="mssql_repl-2147200941"></a>MSSQL_REPL-2147200941
[!INCLUDE [SQL Server SQL MI](../../includes/applies-to-version/sql-asdbmi.md)]
    
## <a name="message-details"></a>Détails du message  
  
|Attribut|Valeur|  
|-|-|  
|Nom du produit|SQL Server|  
|ID de l’événement|-2147200941|  
|Source de l’événement|MSSQLServer|  
|Nom symbolique||  
|Texte du message|Le processus de fusion a détecté une incohérence lors de l'évaluation de l'expression de validation de la partition d'abonné. Ce problème peut être résolu en réinitialisant l'abonnement.|  
  
## <a name="explanation"></a>Explication  
 Le processus de fusion a détecté une incompatibilité pendant qu'il évaluait l'expression de validation de la partition de l'Abonné.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Réinitialisez l'abonnement.
