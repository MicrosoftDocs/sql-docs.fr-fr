# <a name="close-a-solution"></a>Fermer une solution
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
Vous pouvez fermer une solution et la retirer de l'Explorateur de solutions. Fermer une solution la rend inutilisable dans l'Explorateur de solutions.  
  
> [!NOTE]  
> La fermeture de l'Explorateur de solutions n'entraîne pas celle de la solution. Elle masque uniquement la fenêtre de l'Explorateur de solutions.  
  
### <a name="to-close-a-solution"></a>Pour fermer une solution  
  
1.  Dans le menu **Fichier** , cliquez sur **Fermer la solution**.
