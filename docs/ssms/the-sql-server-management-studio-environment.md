# <a name="the-sql-server-management-studio-environment"></a>Environnement SQL Server Management Studio
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
[!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] offre un environnement performant pour la gestion et le développement de requêtes dans [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)].  

> [!NOTE]
> Des conflits avec les pilotes graphiques ont été détectés comme provoquant des problèmes dans le processus de rendu SQL Server Management Studio et, en tant que telle, cette accélération graphique matérielle est désactivée par défaut.  Si vous rencontrez un rendu ou d’autres problèmes visuels dans l’interface SSMS, vérifiez que l’option « Utiliser l’accélération graphique matérielle si elle est disponible » est *désactivée* dans Outils > Options.
