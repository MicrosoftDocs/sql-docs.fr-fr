# <a name="msreportserver_instance-properties---editionname"></a>Propriétés MSReportServer_Instance - EditionName
  Retourne le nom explicite de l'édition du serveur de rapports. Lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## <a name="property-value"></a>Valeur de propriété  
 Chaîne ( **string** ) qui contient le nom lisible de l’édition du serveur de rapports.
