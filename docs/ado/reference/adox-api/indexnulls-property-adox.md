# <a name="indexnulls-property-adox"></a>IndexNulls, propriété (ADOX)
Indique si les enregistrements qui ont des valeurs NULL dans leurs champs d’index ont des entrées d’index.  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit et retourne une valeur [AllowNullsEnum](../../../ado/reference/adox-api/allownullsenum.md) . La valeur par défaut est **adIndexNullsDisallow**.  
  
## <a name="remarks"></a>Notes  
 Cette propriété est en lecture seule sur les objets d' [index](../../../ado/reference/adox-api/index-object-adox.md) déjà ajoutés à une collection.
