# <a name="type-property-column-adox"></a>Type, propriété (colonne) (ADOX)
Indique le type de données d’une colonne.  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit ou retourne une valeur de **type long** qui peut être l’une des constantes [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) . La valeur par défaut est **adVarWChar**.  
  
## <a name="remarks"></a>Notes  
 Cette propriété est en lecture/écriture jusqu’à ce que l’objet de [colonne](../../../ado/reference/adox-api/column-object-adox.md) soit ajouté à une collection ou à un autre objet, après quoi il est en lecture seule.
