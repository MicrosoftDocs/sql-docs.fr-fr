# <a name="type-property-key-adox"></a>Type, propriété (clé) (ADOX)
Indique le type de la clé.  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit ou retourne une valeur de **type long** qui peut être l’une des constantes [KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md) . La valeur par défaut est **adKeyPrimary**.  
  
## <a name="remarks"></a>Notes  
 Cette propriété est en lecture seule sur les objets [clés](../../../ado/reference/adox-api/key-object-adox.md) déjà ajoutés à une collection.
