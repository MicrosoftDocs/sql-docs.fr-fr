# <a name="reserved-database-names-in-parallel-data-warehouse"></a>Noms des bases de données réservées en parallèle Data Warehouse
Un nom de base de données réservé est un nom qui ne peut pas être affecté à une base de données utilisateur dans SQL Server PDW.  
  
Liste des noms de base de données réservés :  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
