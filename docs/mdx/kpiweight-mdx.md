# <a name="kpiweight-mdx"></a>KPIWeight (MDX)


  Retourne le poids de l'indicateur de performance clé spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>Arguments  
 *KPI_Name*  
 Expression de chaîne valide qui précise le nom de l'indicateur de performance clé.  
  
## <a name="remarks"></a>Notes  
 La valeur retournée est la contribution du KPI au parent.
