# <a name="calculationcurrentpass-mdx"></a>CalculationCurrentPass (MDX)


  Retourne le test de calcul actuel d'un cube pour le contexte de requête spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
CalculationCurrentPass()  
```  
  
## <a name="remarks"></a>Notes  
 La fonction **CalculationCurrentPass** retourne l’index de base zéro du test de calcul pour le contexte de requête actuel. Avec la résolution de récurrence automatique, cette fonction n’a que peu d’utilité.
