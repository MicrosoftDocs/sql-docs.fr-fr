**Conformité**  
 Version introduite : conformité des normes ODBC 1,0 : déconseillé  
  
 **Résumé**  
 Dans ODBC 3. *x*, ODBC 2. la fonction *x* **SQLAllocConnect** a été remplacée par **SQLAllocHandle**. Pour plus d’informations, consultez [fonction SQLAllocHandle](../../../odbc/reference/syntax/sqlallochandle-function.md).
