# <a name="jet-outer-joins"></a>Jet : Jointures externes
Une instruction SELECT peut contenir une liste de clauses de jointure externe. Les JOINTUREs externes imbriquées sont prises en charge. Pour plus d’informations sur les JOINTUREs externes, consultez le *Guide du programmeur Microsoft Jet moteur de base de données*.
