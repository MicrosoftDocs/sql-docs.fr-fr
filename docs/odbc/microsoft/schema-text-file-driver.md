# <a name="schema-text-file-driver"></a>Schéma (pilote du fichier texte)
Le schéma inclut des informations sur chaque table (fichier texte) dans une source de données, y compris le format de la table, le nombre de lignes à analyser pour déterminer les types de colonne, si la première ligne de la table contient des noms de colonnes, si le fichier source est écrit à l’aide d’une page de codes OEM
