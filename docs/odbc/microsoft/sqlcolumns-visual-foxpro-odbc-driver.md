# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau 1  
  
 Pour une table, crée un jeu de résultats qui est la liste des colonnes de la ou des tables spécifiées.  
  
 Pour plus d’informations, consultez [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) dans le *Guide de référence du programmeur ODBC*.
