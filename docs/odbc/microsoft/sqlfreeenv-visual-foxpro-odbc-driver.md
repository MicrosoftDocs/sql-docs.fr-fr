> [!NOTE]

 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau principal  
  
 Ferme le pilote ODBC Visual FoxPro et libère toute la mémoire associée au pilote.  
  
 Pour plus d’informations, consultez [sqlfreeenv,](../../odbc/reference/syntax/sqlfreeenv-function.md) dans le *Guide de référence du programmeur ODBC*.
