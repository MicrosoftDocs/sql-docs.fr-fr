# <a name="sqlallocenv-visual-foxpro-odbc-driver"></a>SQLAllocEnv (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous référence de l' [API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité de l’API ODBC : niveau principal  
  
 Alloue de la mémoire pour un handle d’environnement, *henv*, et initialise l’interface de niveau d’appel ODBC pour une utilisation par une application.  
  
 Pour plus d’informations, consultez [SQLAllocEnv](../../odbc/reference/syntax/sqlallocenv-function.md) dans le *Guide de référence du programmeur ODBC*.
